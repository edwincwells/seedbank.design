How trustworthy a claim is — verified, self-reported, or not disclosed. Composes the badge component. The green on "verified" is self-policing: it appears only when a rationale link is present. This page shows each state styled by verification-status.css.

A "verified" claim with no linked evidence renders as the neutral badge above, not green. This is intentional: a green verified badge without a rationale link is non-conforming under principle 02, and the CSS refuses to grant the green. Either add the rationale link, or use self-reported if the claim genuinely is unverified.
