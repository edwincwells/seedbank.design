table

A semantic data table for read-only display. Editorial visual treatment — hairline horizontal row separators, no vertical lines, no zebra striping. Optional row-level status via activity-coloured left-rules. Wrap in <div class="table-wrap"> for horizontal scroll at narrow viewports.

avoid — don't use this component for layout, sortable grids, or selectable rows

The table component is for semantic display of data. If you need sorting, filtering, row selection, or pagination, build those behaviours on top of this component — they are not variants of it. If you're using a <table> for page layout, stop; use a grid or flex layout instead.
